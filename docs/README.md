# DayTrace Docs Index

Краткий индекс основных документов проекта (что читать и для чего).

- `../README.md` — стартовая точка: обзор проекта, локальный запуск, Docker Compose, env-переменные, эксплуатационные заметки.
- `PRD.md` — продуктовые требования, API-контракты и acceptance criteria (источник требований).
- `METRICS.md` — definitions/SQL для продуктовых метрик и проверки качества данных.
- `RUNTIME_WORKERS.md` — фоновые `HostedService` в API: интервалы, триггеры, зависимости от env, надежность.
- `../CLAUDE.md` — инженерные/архитектурные заметки и workflow для AI-агентов и контрибьюторов.
